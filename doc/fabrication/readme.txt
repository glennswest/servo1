File Extension	Meaning"G1, G2, etc"	"Mid-layer 1, 2, etc"GBL	Bottom LayerGBO	Bottom OverlayGBP	Bottom Paste MaskGBS	Bottom Solder Mask"GD1, GD2, etc"	Drill Drawing (assignment based on order of drill pairs appearing in the Drill-Pair Manager dialog)"GG1, GG2, etc"	Drill Guide (assignment based on order of drill pairs appearing in the Drill-Pair Manager dialog)GKO	Keep Out Layer"GM1, GM2, etc"	"Mechanical Layer 1, 2, etc""GP1, GP2, etc"	"Internal Plane Layer 1, 2, etc"GPB	Pad Master BottomGPT	Pad Master TopGTL	Top LayerGTO	Top OverlayGTP	Top Paste MaskGTS	Top Solder Mask"P01, P02, etc"	Gerber PanelsAPR	Aperture File (generated when embedded apertures (RS274X) are used)APT	Aperture File (generated when embedded apertures (RS274X) are not used)
